# OverTheWire Bandit Writeups

A collection of my solutions, notes, and learning journey through the **OverTheWire Bandit** wargame.

The goal of this repository is to document my progress while improving my Linux command-line skills, scripting knowledge, and problem-solving abilities required for cybersecurity and CTF competitions.

## About Bandit

Bandit is a beginner-friendly Linux wargame created by **OverTheWire**. It teaches fundamental Linux concepts through a series of progressively challenging levels.

You'll learn topics such as:

- Linux file system navigation
- File permissions
- Hidden files
- SSH
- Text processing
- Pipes and redirection
- Compression
- Encoding
- Networking basics
- Scripting fundamentals

Official Website:
https://overthewire.org/wargames/bandit/
